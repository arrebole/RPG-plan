### #人物系统

#### (一)属性点

>1、升级获取属性点，属性点自由分配。
>
>3、属性值为基础属性，不同装备收益不同



| 属性   | 效果         | 适用     |
| ------ | ------------ | -------- |
| 生命力 | 提升 maxHP   |          |
| 精力   | 提升 maxMP   | 技能输出 |
| 力量   | 物理攻击     | 物理攻击 |
| 智力   | 魔法伤害     | 法术攻击 |
| 信仰   | 奇迹技能     | 奶妈     |
| 运气   | 影响各种效果 | 高爆发   |

例子：

​	力量20的人物装备1把长剑，力量收益 50%。

​	人物 物理面板 = 80 + 20 *50%

同一把武器可能会有多种属性收益，属性点超出30后收益下降。

>2、技能来源：升级获得、技能书学习。

>3、技能通过熟练度升级。

分类

+ 物理系:  战技类
+ 魔法系：元素类，召唤类，等
+ 奇迹系:  治愈类，buff类
+ ex：从未见过的技能

### #战斗系统

#### (一)Buff状态（再说）

>1、正面buf，负面bufff
>
>2、特定装备或技能或称号可以免疫buff。

例如

+ 正面buff ：勇气，
+ 负面buff：出血，胆怯、



#### (二)伤害计算

>攻击：物理攻击、法术攻击
>
>护甲：物理防御、法术防御
>
>暴击：基础暴击伤害加成 * 1.5

 例如：

| 武器 | 基础 | 收益     | 穿透 | 其他效果 |
| ---- | ---- | -------- | ---- | -------- |
| 长剑 | 80   | +50%力量 | 0    | 无       |

伤害 =  （80 + 力量 x 50%）x（其他增益）- 物防 x (1-穿透)

对防御高的无伤害。



#### (三)装备掉落

>安全区外红名，死亡掉落装备

对中立生物造成伤害，会提升罪恶值，罪恶值达到100，进入红名状态。

人物死亡清空所有罪恶值，随机掉落装备。

### #商店系统

>流通货币：金币和银币
>
>金币可以转换为银币，银币暂时无法转换。
>
>1RMB = 10金币



#### 标准商店

武器只能通过金币购买。银币用于购买材料。

商店可以回收(特定)道具，返回银币。

金币可以通过任务，击败boss，活动获取。



#### 自由商店

允许玩家出售装备给其他玩家，价格在规定范围内，交易为公开，全服可抢。

一旦出售优先购买者获得交易权。

玩家摊位有限。



### #工匠系统

>部分武器可以通过材料打造。
>
>低级材料可以合成高级材料。
>

武器 = 材料+少量金币；

药 = 材料 + 银币

染色 = 染料 + 特定武器 + 金币；
